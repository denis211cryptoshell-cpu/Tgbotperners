## 🚀 Установка и запуск

### 1. Получите токен бота

1. Откройте @BotFather в Telegram
2. Отправьте команду `/newbot`
3. Придумайте имя и username бота
4. Скопируйте полученный токен

### 2. Настройте конфигурацию

Создайте файл `.env` по примеру `.env.example`:

```bash
TOKEN=ваш_токен_здесь
ADMIN_ID=ваш_telegram_id
```

**Как узнать свой Telegram ID:**
- Напишите боту @userinfobot
- Или посмотрите в настройках Telegram (если есть)

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустите бота

```bash
python bot.py
```

## 🐳 Docker

### Запуск с Docker Compose (рекомендуется)

```bash
# Сборка и запуск
docker-compose up -d

# Просмотр логов
docker-compose logs -f

# Остановка
docker-compose down
```

### Запуск с обычным Docker

```bash
# Сборка образа
docker build -t tg-bot-tpartners .

# Запуск контейнера
docker run -d \
  --name tg-bot \
  --env-file .env \
  -v bot_data:/app/data \
  tg-bot-tpartners
```

## 📊 Админ-панель

### Доступ к админ-панели

Отправьте боту команду `/admin`

**Важно:** Доступно только пользователю с `ADMIN_ID` из `.env`

### Функции админ-панели:

| Функция | Описание |
|---------|----------|
| 📊 Статистика | Пользователи, активные за 7 дней, конверсии |
| 📈 По продуктам | Клики по каждому продукту (Black, кредитка, инвестиции) |
| 🔄 Воронки | Конверсия каждой воронки (сколько дошло до конца) |
| 📥 Экспорт | Выгрузка всех пользователей в CSV |
| 📤 Рассылка | Отправка сообщений всем пользователям |

### Пример отчёта статистики:

```
📊 Статистика бота

👥 Пользователи:
   • Всего: 150
   • Активных за 7 дней: 45

🎯 Конверсии (переходы по ссылкам):
   • Всего: 89

📈 По продуктам:
   💳 black: 35
   💰 credit_card: 28
   📈 investment: 18
   📱 t_mobile: 5
   🛡 insurance: 3

🔄 Воронки:
   • black: 28/35 (80.0%)
   • credit_card: 22/28 (78.6%)
   • investment: 15/18 (83.3%)
```

## 🗄️ База данных

Бот использует SQLite базу данных (`bot_database.db`).

### Таблицы:

| Таблица | Описание |
|---------|----------|
| `users` | Пользователи (ID, имя, дата регистрации, последний вход) |
| `actions` | Действия пользователей (какие кнопки нажимали) |
| `conversions` | Переходы по ссылкам (конверсии) |
| `funnels` | Пройденные этапы воронок |
| `mailings` | История рассылок |

### Автоматический сбор данных:

- ✅ Каждый пользователь сохраняется при первом запуске
- ✅ Все нажатия кнопок логируются
- ✅ Переходы по ссылкам записываются
- ✅ Этапы воронок сохраняются
- ✅ `last_seen_at` обновляется при каждом действии

## 🧮 Калькулятор кэшбэка

Новая функция для повышения конверсии!

**Как работает:**
1. Пользователь вводит свои месячные траты
2. Бот считает кэшбэк с обычной карты (1%)
3. Бот считает кэшбэк с Tinkoff Black (до 15%)
4. Показывает разницу и годовую выгоду

**Пример:**
```
Траты в месяц: 50 000 ₽

💳 Стандартная карта (1%):
   • В месяц: 500 ₽
   • В год: 6 000 ₽

🔥 Tinkoff Black (до 15%):
   • В месяц: 4 000 ₽
   • В год: 48 000 ₽

💰 Ваша выгода:
   • В месяц: +3 500 ₽
   • В год: +42 000 ₽
```

## 📊 Мониторинг и аналитика

### Что отслеживается:

1. **Пользователи:**
   - Всего зарегистрировано
   - Активных за последние 7 дней
   - Дата первого запуска
   - Дата последнего визита

2. **Конверсии:**
   - Переходы по ссылкам на продукты
   - Конверсия по каждому продукту

3. **Воронки:**
   - Сколько начали прохождение
   - Сколько дошли до конца
   - На каком этапе отвалились

4. **Действия:**
   - Какие кнопки нажимают
   - Какие команды используют
